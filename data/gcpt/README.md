## Key Fields

| Field | Description |
|-------|-------------|
| Project Name | Name of the power plant |
| Unit Name | Name of the specific unit |
| Country/Area | Country where plant is located |
| Subnational | State/province/region |
| Latitude | Plant latitude (WGS84) |
| Longitude | Plant longitude (WGS84) |
| Status | Operating, Announced, Pre-permit, etc. |
| Capacity (MW) | Nameplate capacity in megawatts |
| EIA plant + unit ID | US EIA plant and unit identifier (format: "plant_id|unit_id") |
